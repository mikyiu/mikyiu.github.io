# My Unhinged Web Project

Hello, this is a website I wanted to make for myself. I wanted to make this with REACT to get some practice in! This website is for my own personal use, not professional use, no money is being made with this! 


## 🛠️ The Tech Stack
* **Framework:** React Native Web (via Expo)
* **Routing:** Expo Router (File-based routing)
* **Styling:** Standard React Native StyleSheets

## 📦 Packages Used
* `expo-linear-gradient`: For the deep space void background.
* `expo-blur`: For frosted glass (currently benched for authentic Win95 gray, but kept in the arsenal).
* `@react95/icons`: Providing the open-source Windows 95 icon set.
